Privacy Policy

Last updated: April 4, 2026  ·  DevResolve, Inc.

1. Overview

DevResolve, Inc. ("DevResolve", "we", "us") operates the DevResolve autonomous ticket deflection platform accessible at devresolve.ai and api.devresolve.ai. This Privacy Policy explains what data we collect, how we use it, and your rights.

2. Zero-Knowledge Architecture

DevResolve is built on a zero-knowledge principle. When a ticket is processed:

  • Ticket content, Slack thread messages, and knowledge source results are loaded into RAM only for the duration of the AI synthesis call.
  • This content is explicitly deleted from memory after synthesis completes.
  • No ticket body text, customer messages, or knowledge source content is written to our database, vector store, or logs.
  • Your data is never used to train AI models.

3. What We Store

Our database retains only metadata — never content:

  • Organisation record: company name, plan tier, webhook secret (hashed), Stripe customer ID.
  • Resolution record per ticket: ticket ID (e.g. "DEV-123"), confidence score, resolution status (success / pending_review / escalated), billing event ID, timestamps.
  • Knowledge source configuration: connector type (Confluence, Notion, etc.), API credentials (stored encrypted), space keys.

We do not store ticket summaries, descriptions, customer names, email addresses, or any personally identifiable information from processed tickets.

4. Atlassian Integration

When you connect DevResolve to Jira via webhook, DevResolve receives Jira webhook payloads containing ticket keys and metadata. This data is processed in real time and not persisted beyond the metadata described in Section 3. DevResolve requests only the minimum Jira permissions required to read issue data and post comments.

5. Third-Party Services

  • Anthropic (Claude): Ticket context is sent to Anthropic's API for AI synthesis. Anthropic's data processing agreement applies. Data is not retained by Anthropic beyond the API call per their zero-retention policy for API customers.
  • Stripe: Payment and billing data is handled by Stripe, Inc. DevResolve does not store payment card information.
  • Pinecone: Ingested documentation vectors are stored in Pinecone namespaced per organisation. Raw document text is not stored — only vector embeddings.

6. Data Retention

Resolution metadata is retained for the duration of your subscription and deleted within 30 days of account closure. Knowledge source credentials are deleted immediately upon disconnection.

7. Security

  • All webhook requests are authenticated via HMAC-SHA256 signature verification.
  • All data in transit is encrypted via TLS 1.2+.
  • Database access is restricted to application containers on a private network.
  • SSH access to production servers requires key-based authentication only.

8. Your Rights

You may request deletion of your organisation's data at any time by contacting us. Because we store only metadata (no ticket content or personal data from tickets), deletion requests are fulfilled within 7 business days.

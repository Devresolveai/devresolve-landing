How it works

Connect your tools. Your AI gets a brain.

Four steps. The first one takes 20 minutes. The rest is automatic.

01 — Connect

Plug in your knowledge sources

Confluence, Notion, Jira, Zendesk, GitHub, any docs URL. Self-serve setup. We pull it all in and index it.

02 — Synthesize

Unified knowledge layer

API

Built for developers. Not for demos.

REST API and MCP server. Ship in an afternoon.

POST/api/v1/query

Ask a question. Get a synthesized answer with confidence score and source citations.

POST/api/v1/ingest

Feed it a URL or raw text. Content gets chunked, embedded, and indexed automatically.

POST/api/v1/search

Raw vector search. Returns matching document chunks with relevance scores.

GET/api/v1/sources

List all connected knowledge sources for your org.

DEL/api/v1/sources/{id}

Remove a knowledge source.
